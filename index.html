Formal Set Theory

Our project is to formalise and check the presentation of elementary axiomatic set theory found in the book General Topology by John Kelley (Van Nostrand, 1955). The section on set theory is found on pp.250-281. This axiomatic system is close to the von Neumann-Bernays-GÃ¶del one and is sometimes called Kelley-Morse set theory.
