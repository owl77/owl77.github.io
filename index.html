Formal Categorical Logic

Formalizing categorical logic and type theory in Coq.


Homotopy Type Theory Basics

Homotopy Type Theory is built over Martin-LÃ¶f type theory with (higher) inductive types. There are three kinds of judgment:

      \( \Gamma\) ctx            \(\Gamma \vdash a: A\)             \( \Gamma \vdash a \equiv a': A\)

A context is a possibly empty list \(x_1 :A_1, x_2: A_2,...,x_m:A_n\) in which the variables \(x_1,...,x_n\) are all distinct and the \(A_1,A_2,...,A_n\) are types. We use \(\cdot\) to denote the empty context. There is really no distinction between terms and types unlike in weaker type systems. Types are certain kinds of terms. We now specify the syntax of terms. Our presentation will be geared towards implementing a proof assistant and proof checker. For a given environment we will have a list of defined constants \(f,f',f'',...\) along with the built-in primitive constants \(c,c',c',...\). Terms are defined as usual as untyped \(\lambda\)-terms:

          \( t::= x \quad |\quad \lambda x.t\quad |\quad tt' \quad |\quad c \quad |\quad f \)

We have primitive constants \(\mathcal{U}_0,\, \mathcal{U}_1,\, \mathcal{U}_2,...\) which are a cumulative hierarchy of universes. The rules of inference for judgments \(\Gamma\) ctx are the following

\( \begin{prooftree} \AxiomC{$$} \RightLabel{ctx-Emp} \UnaryInfC{$\cdot\quad ctx$} \quad \AxiomC{$x_1:A_1,...,x_{n-1}: A_{n-1} \vdash A_n: \mathcal{U}_n$} \RightLabel{ctx-Ext} \UnaryInfC{$(x_1:A_1,...,x_n:A_n) \text{ ctx}$} \AxiomC{$(x_1:A_1,...,x_n:A_n) \quad \text{ctx}$} \RightLabel{Vble} \UnaryInfC{$x_1:A_1,...,x_n:A_n \vdash x_i: A_i$} \end{prooftree} \)

For type universes:

\( \begin{prooftree} \AxiomC{$\Gamma\quad \text{ctx}$} \RightLabel{$\mathcal{U}$-Intro } \UnaryInfC{$\Gamma\vdash \mathcal{U_i}:\mathcal{U_{i+1}}$} \AxiomC{$\Gamma\vdash A: \mathcal{U_i}$} \RightLabel{$\mathcal{U}$-Cumul} \UnaryInfC{$\Gamma \vdash A: \mathcal{U}_{i+1}$} \end{prooftree} \)

Judgmental equality is an equivalence relation respected by typing:

\( \begin{prooftree} \AxiomC{$\Gamma \vdash a:A$} \RightLabel{ } \UnaryInfC{$\Gamma \vdash a\equiv a:A$} \AxiomC{$\Gamma \vdash a \equiv b :A$} \RightLabel{ } \UnaryInfC{$\Gamma \vdash b \equiv a: A$} \AxiomC{$\Gamma \vdash a\equiv b:A$} \AxiomC{$\Gamma \vdash b \equiv c:A$} \RightLabel{ } \BinaryInfC{$\Gamma\vdash a \equiv c: A$} \AxiomC{$\Gamma \vdash a: A$} \AxiomC{$\Gamma \vdash A\equiv B:\mathcal{U}_i$} \RightLabel{ } \BinaryInfC{$\Gamma \vdash a:B$} \AxiomC{$\Gamma \vdash a \equiv b: A$} \AxiomC{$\Gamma\vdash A\equiv B:\mathcal{U}_i$} \RightLabel{ } \BinaryInfC{$\Gamma \vdash a \equiv b:B$} \end{prooftree} \)

Dependent Types:

\( \begin{prooftree} \AxiomC{$\Gamma \vdash A:\mathcal{U}_i$} \AxiomC{$\Gamma, x:A \vdash B:\mathcal{U}_i$} \RightLabel{$\Pi$-Form } \BinaryInfC{$\Gamma \vdash \Pi_{(x:A)} B:\mathcal{U}_i$} \AxiomC{$\Gamma, x:A \vdash f(x) : B$} \RightLabel{$\Pi$-Intro } \UnaryInfC{$\Gamma \vdash \lambda(x:A) f(x): \Pi_{(x:A)} B$} \AxiomC{$\Gamma \vdash f: \Pi_{(x:A)} B$} \AxiomC{$\Gamma \vdash a:A$} \RightLabel{$\Pi$-Elim } \BinaryInfC{$\Gamma \vdash fa: B[a/x]$} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, x:A \vdash f(x): B$} \AxiomC{$\Gamma \vdash a:A$} \RightLabel{$\Pi$-Comp } \BinaryInfC{$\Gamma \vdash (\lambda(x:A)f(x))a \equiv f(a) : B$} \AxiomC{$\Gamma \vdash f: \Pi_{(x:A) B}$} \RightLabel{$\Pi$-Uniq } \UnaryInfC{$\Gamma \vdash f \equiv \lambda(x:A) f(x): \Pi_{(x:A)} B$} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma \vdash A:\mathcal{U_i}$} \AxiomC{$\Gamma, x: A \vdash B : \mathcal{U_i}$} \RightLabel{$\Sigma$-Form } \BinaryInfC{$\Gamma \vdash \Sigma_{(x:A)} B: \mathcal{U_i} $} \AxiomC{$\Gamma, x:A \vdash B:\mathcal{U_i}$} \AxiomC{$\Gamma \vdash a : A$} \AxiomC{$\Gamma \vdash b : B[a/x]$} \RightLabel{$\Sigma$-Intro } \TrinaryInfC{$\Gamma \vdash (a,b): \Sigma_{(x:A)} B$} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, z: \Sigma_{x:A} B \vdash C:\mathcal{U_i}$} \AxiomC{$\Gamma, x:A, y:B \vdash g : C[(x,y)/ z]$} \AxiomC{$\Gamma \vdash p : \Sigma_{x:A} B$} \RightLabel{$\Sigma$-Elim } \TrinaryInfC{$\Gamma \vdash \text{ind}(\Sigma_{(x:A)}B, \lambda z. C, \lambda x,y.g, p) : C[ p/z ]$} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, z: \Sigma_{x:A} B \vdash C:\mathcal{U_i}$} \AxiomC{$\Gamma, x:A, y:B \vdash g : C[(x,y)/ z]$} \AxiomC{$\Gamma \vdash a : A$} \AxiomC{$\Gamma \vdash b : B[a/x]$} \RightLabel{$\Sigma$-Comp } \QuaternaryInfC{$\Gamma \vdash \text{ind}(\Sigma_{(x:A)}B, \lambda z. C, \lambda x,y.g, (a,b)) \equiv g[a,b/ x, y]: C[ (a,b)/z ]$} \end{prooftree} \)

Coproducts:

\( \begin{prooftree} \AxiomC{$\Gamma \vdash A:\mathcal{U_i}$} \AxiomC{$\Gamma \vdash B: \mathcal{U_i}$} \RightLabel{$+$-Form } \BinaryInfC{$ \Gamma \vdash A + B : \mathcal{U_i}$} \AxiomC{$\Gamma \vdash A:\mathcal{U_i}$} \AxiomC{$\Gamma \vdash B: \mathcal{U_i}$} \AxiomC{$\Gamma \vdash a : A$} \RightLabel{$+$-IntroL } \TrinaryInfC{$ \Gamma \vdash \text{inl}(a): A + B$} \AxiomC{$\Gamma \vdash A:\mathcal{U_i}$} \AxiomC{$\Gamma \vdash B: \mathcal{U_i}$} \AxiomC{$\Gamma \vdash b : B$} \RightLabel{$+$-IntroR } \TrinaryInfC{$ \Gamma \vdash \text{inr}(b): A + B$} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, z : A + B \vdash C: \mathcal{U_i}$} \AxiomC{$\Gamma, x:A \vdash a: C[\text{inl}(x)/z]$} \AxiomC{$\Gamma, y:B \vdash b: C[\text{inl}(y)/z]$} \AxiomC{$\Gamma \vdash c : A + B$} \RightLabel{$+$-Elim } \QuaternaryInfC{$ \Gamma \vdash \text{ind}(A+B, \lambda x. a, \lambda y. b, c ) : C[ c/z] $} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, z : A + B \vdash C: \mathcal{U_i}$} \AxiomC{$\Gamma, x:A \vdash a: C[\text{inl}(x)/z]$} \AxiomC{$\Gamma, y:B \vdash b: C[\text{inl}(y)/z]$} \AxiomC{$\Gamma \vdash c : A$} \RightLabel{$+$-Elim } \QuaternaryInfC{$ \Gamma \vdash \text{ind}(A+B, \lambda x. a, \lambda y. b, \text{inl}(c) ) \equiv a[c/x] : C[ \text{inl}(c)/z] $} \end{prooftree} \)

Zero and Unit Types:

\( \begin{prooftree} \AxiomC{$\Gamma \quad\text{ctx}$} \RightLabel{$\mathbf{0}$-Form } \UnaryInfC{$ \Gamma \vdash \mathbf{0}: \mathcal{U_i}$} \AxiomC{$\Gamma , x:\mathbf{0} \vdash C: \mathcal{U_i}$} \AxiomC{$\Gamma \vdash a:\mathbf{0}$} \RightLabel{$\mathbf{0}$-Elim} \BinaryInfC{$ \Gamma \vdash \text{ind}(\mathbf{0},\lambda x.C,a): C[a/x]$} \AxiomC{$\Gamma \quad\text{ctx}$} \RightLabel{$\mathbf{1}$-Form } \UnaryInfC{$ \Gamma \vdash \mathbf{1}: \mathcal{U_i}$} \AxiomC{$\Gamma \quad\text{ctx}$} \RightLabel{$\mathbf{1}$-Intro } \UnaryInfC{$ \Gamma \vdash \star:\mathbf{1}$} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma , x:\mathbf{1} \vdash C: \mathcal{U_i}$} \AxiomC{$\Gamma \vdash c: C[\star /x]$} \AxiomC{$\Gamma \vdash a:\mathbf{1}$} \RightLabel{$\mathbf{1}$-Elim } \TrinaryInfC{$ \Gamma \vdash \text{ind}(\mathbf{1},\lambda x.C,c,a): C[a/x]$} \AxiomC{$\Gamma , x:\mathbf{1} \vdash C: \mathcal{U_i}$} \AxiomC{$\Gamma \vdash c: C[\star /x]$} \RightLabel{$\mathbf{1}$-Comp} \BinaryInfC{$ \Gamma \vdash \text{ind}(\mathbf{1},\lambda x.C,c,\star) \equiv \star: C[\star/x]$} \end{prooftree} \)

Natural Number Type:

\( \begin{prooftree} \AxiomC{$\Gamma \quad\text{ctx}$} \RightLabel{$\mathbb{N}$-Form } \UnaryInfC{$\Gamma \mathbb{N}:\mathcal{U_i} $} \AxiomC{$\Gamma \vdash \mathbb{N}: \mathcal{U_i} $} \RightLabel{$\mathbb{N}$-Intro$_0$ } \UnaryInfC{$ \Gamma \vdash 0: \mathbb{N}$} \AxiomC{$\Gamma \vdash n: \mathbb{N} $} \RightLabel{$\mathbb{N}$-Intro$_1$ } \UnaryInfC{$ \Gamma \vdash succ(n): \mathbb{N}$} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, x: \mathbb{N} \vdash C: \mathcal{U_i} $} \AxiomC{$\Gamma \vdash c_0 :C[0/x]$} \AxiomC{$\Gamma, x: \mathbb{N}, y : C \vdash c: C[succ(x)/x] $} \AxiomC{$\Gamma \vdash n: \mathbb{N} $} \RightLabel{$\mathbb{N}$-Elim } \QuaternaryInfC{$\Gamma \vdash \text{ind}(\mathbb{N},\lambda x.C, c_0, \lambda x,y. c, n): C[n/x] $} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, x: \mathbb{N} \vdash C: \mathcal{U_i} $} \AxiomC{$\Gamma \vdash c_0 :C[0/x]$} \AxiomC{$\Gamma, x: \mathbb{N}, y : C \vdash c: C[succ(x)/x] $} \RightLabel{$\mathbb{N}$-Comp$_0$ } \TrinaryInfC{$\Gamma \vdash \text{ind}(\mathbb{N}, \lambda x.C, c_0, \lambda x,y. c, 0) \equiv c_0: C[0/x] $} \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, x: \mathbb{N} \vdash C: \mathcal{U_i} $} \AxiomC{$\Gamma \vdash c_0 :C[0/x]$} \AxiomC{$\Gamma, x: \mathbb{N}, y : C \vdash c: C[succ(x)/x] $} \AxiomC{$\Gamma \vdash n: \mathbb{N} $} \RightLabel{$\mathbb{N}$-Comp$_1$ } \QuaternaryInfC{$\Gamma \vdash \text{ind}(\mathbb{N},\lambda x.C, c_0, \lambda x,y. c, succ(n)) \equiv c[n, \text{ind}(\mathbb{N}, \lambda x.C, c_0, \lambda x,y. c, n)/ x,y ] : C[succ(n)/x] $} \end{prooftree} \)

Identity Types:

\( \begin{prooftree} \AxiomC{$\Gamma \vdash A:\mathcal{U_i} $} \AxiomC{$\Gamma \vdash a: A$} \AxiomC{$\Gamma \vdash b: A$} \RightLabel{$=$-Form } \TrinaryInfC{$\Gamma \vdash a=_A b: \mathcal{U_i}$ } \AxiomC{$\Gamma \vdash A:\mathcal{U_i} $} \AxiomC{$\Gamma \vdash a: A$} \RightLabel{$=$-Const } \BinaryInfC{$\Gamma \vdash \text{ref}_a :a=_A a$ } \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, x:A, y:A, p: x=_A y \vdash C:\mathcal{U_i} $} \AxiomC{$\Gamma \vdash a: A$} \AxiomC{$\Gamma \vdash b: A$} \AxiomC{$\Gamma \vdash p': a_A = b$} \AxiomC{$\Gamma, z:A \vdash c: C[z,z,\text{refl}_z /x,y,p]$} \RightLabel{$=$-Elim } \QuinaryInfC{$\Gamma \vdash \text{ind}_=(\lambda x,y,p.C, \lambda z.c, p') : C[a,b,p' /x,y,p] $ } \end{prooftree} \)

\( \begin{prooftree} \AxiomC{$\Gamma, x:A, y:A, p: x=_A y \vdash C:\mathcal{U_i} $} \AxiomC{$\Gamma \vdash a: A$} \AxiomC{$\Gamma, z:A \vdash c: C[z,z,\text{refl}_z /x,y,p]$} \RightLabel{$=$-Comp } \TrinaryInfC{$\Gamma \vdash \text{ind}_=(\lambda x,y,p.C, \lambda z.c, \text{refl}_a)\equiv c[a /z] : C[a,a,\text{refl}_a /x,y,p] $ } \end{prooftree} \)

Consider \(f,g: \Pi_{(x:A)} P(x)\). Then a homotopy from f to g is a term of type \(\Pi_{(x:A)} f(x) = g(x)\) which we denote by \(f \sim g\). One definition of the type \(\text{isequiv}(f) \) is \( \Sigma_{g:B\rightarrow A}f\circ g \sim id_B \times \Sigma_{h:B\rightarrow A}h\circ f \sim id_A\). The type of equivalences \(A\simeq B\) is given by \(\Sigma_{f:A\rightarrow B}\text{isequiv}(f)\).

Given \(\Gamma \vdash A:\mathcal{U_i}\) and \(\Gamma \vdash A:\mathcal{U_i}\) we can derive \[ \Gamma, f:\Pi_{(x:A)}B, g: \Pi_{(x: A)} B \vdash \text{happly}(f,g) : (f =_{A\rightarrow B} g)\rightarrow f\sim g \] \[ \Gamma \vdash \text{idtoeqv}(A,B): A =_{\mathcal{U_i}} B \rightarrow A\simeq B \] Extensionality:

\( \begin{prooftree} \AxiomC{$\Gamma \vdash f: \Pi_{(x:A)} B $} \AxiomC{$\Gamma \vdash g: \Pi_{(x:A)} B $} \RightLabel{$\Pi$-Ext } \BinaryInfC{$\Gamma \vdash \text{funext}(f,g) : \text{isequiv}(\text{happly}(f,g)) $ } \end{prooftree} \)

Univalence:

\( \begin{prooftree} \AxiomC{$\Gamma \vdash A:\mathcal{U_i} $} \AxiomC{$\Gamma \vdash B:\mathcal{U_i} $} \RightLabel{$\mathcal{U_i}$-Univ } \BinaryInfC{$\Gamma \vdash \text{univalence}(A,B): \text{isequiv}(\text{idtoeqv}(A,B)) $ } \end{prooftree} \)
